General Machine Learning

Graph Nets
Recommended DeepMind

We witness more and more works on deep learning applied to geometry and graphs. Recently, DeepMind introduced their ambitious Graph Nets in this paper, where they explore how using relational inductive biases within deep learning architectures can facilitate learning about entities, relations, and rules for composing them. This library is provided with well-written, well-explained examples such as the shortest path problem solved by a Graph Net. See also the slides of this introductory tutorial on Graph nets by Yujia Li.

Optimisation & Learning Theory

On the Spectral Bias of Neural Networks
Ruprecht-Karls-Universitat, Mila, Universite de Montreal

[...] By using tools from Fourier analysis, we show that deep ReLU networks are biased towards low frequency functions, meaning that they cannot have local fluctuations without affecting their global behavior. Intuitively, this property is in line with the observation that over-parameterized networks find simple patterns that generalize across data samples. [...]

See also this related work: Training behavior of deep neural networks in frequency domain.

Depth with Nonlinearity Creates No Bad Local Minima in ResNets
MIT, University of Montreal

In this paper, we prove that depth with nonlinearity creates no bad local minima in a type of arbitrarily deep ResNets studied in previous work, in the sense that the values of all local minima are no worse than the global minima values of corresponding shallow linear predictors with arbitrary fixed features, and are guaranteed to further improve via residual representations.

Gans & Adversarial Attacks

Self-Attention GAN
Google Brain

Back in May, the Self-Attention GAN model (SAGAN) was proposed. It combines a GAN with a self-attention map which enables it to generate images using cues from distant locations. This repo is Google Brain's official implementation using TensorFlow.

Reinforcement Learning

Reinforcement Learning with Prediction-Based Rewards
OpenAI

An agent finally managed to beat human performance on Montezuma's Revenge. Interestingly, the approach used here is said to be based on curiosity where the agent is encouraged to explore states it doesn't already know. To do this, the team at OpenAI proposed a variant of their previous work. The agent is trained to predict what will happen in the next state and is rewarded when this prediction turns out to be wrong. In other words: the agent is rewarded when it finds something unexpected! As always with OpenAI's blog posts, you will find plenty of visualisations. The section on the noisy TV problem is quite amusing!

Natural Language Processing

BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding
Google Research

This work made quite a lot of noise in the NLP community recently. Authors propose a new way to obtain word embeddings which is both deep (with long-range context) and bidirectional (the context comes from the previous and next words). With these embeddings, they were able to beat the state-of-the-art on several benchmarks and several dataset. They are releasing the weights which took 4 days to obtain on up to 16 TPUS,... No wonder the GitHub repo has already nearly 4,000 stars.

YSDA Natural Language Processing course
Yandex School of Data Analysis

A repo with plenty of resources to learn more about NLP. Although the video lectures are in Russian, they provide courseworks and Jupyter notebooks in English. Also, for each course material, there is a list of related projects and papers that you can use to deepen your understanding.

Production & Engineering

QNNPACK: Open source library for optimized mobile deep learning
Facebook Code

This blog post has been released to explain how Facebook's library QNNPACK works. It provides convolution operations, transposed convolutions and fully-connected neural nets which are optimised for mobile. The library is already used in Facebook's mobile applications. Their benchmark shows that it is now possible to run models like Mask R-CNN or DensePose in real time!
